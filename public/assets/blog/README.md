# Blog Image Placeholders

This directory should contain the blog cover images referenced in the blog posts.

## Required Images:

1. **render-deployment-cover.jpg**
   - Recommended size: 1200x630px (OpenGraph standard)
   - Suggested content: Screenshot of Render dashboard, or Node.js + Render logos
   - Use tools like Canva, Figma, or Photoshop to create

2. **blog-og-image.jpg**
   - Recommended size: 1200x630px
   - General blog section cover image
   - Should represent your blog/technical content brand

## How to Create Blog Cover Images:

### Quick Options:
1. **Canva** (easiest): Use free templates for blog covers
2. **Unsplash/Pexels**: Free stock photos related to coding/deployment
3. **Carbon.now.sh**: Create beautiful code screenshots

### Image Optimization:
- Compress images using TinyPNG or ImageOptim
- Target file size: < 200KB for fast loading
- Use WebP format for better performance (optional)
